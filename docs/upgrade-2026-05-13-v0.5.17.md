## 保留的本地能力

本次升级遵循“保留本地配置和本地功能，合入上游新功能”的原则，以下能力继续保留：

- `chat.allapple.top` 仍由本机 `8650` 端口提供服务。
- Hermes API Server 仍使用 `0.0.0.0:8642`，未改写为其它端口。
- `AUTH_DISABLED=1` 自动登录配置保留，前端不需要手动输入访问令牌。
- 本地 Liquid Glass / 毛玻璃 UI 适配保留。
- Cloudflare Tunnel 管理页与接口保留。
- Codex 登录、多账号切换与账号删除能力保留。
- 消息页内“账号与模型”快捷切换控件保留。
- Gateway PID / 端口保护逻辑保留，避免 Web UI 重启时误把 Hermes Gateway 从 `8642` 改到其它端口。
- `packages/server/src/routes/index.ts` 仍是实际生效的服务端路由注册入口。
